Hungry Birds Twitter Feed

A class-based ActionScript 3 project that read the twitter feed for Hungry Birds through XML. This project was a joint effort with the talented Stephen Belyea who did the design work while I took care of the programming.

loyl - love of your life

loyl is a mock dating site built on a custom content management system. It allows users to register, including account activation via e-mail. User can also update their profile, upload a profile image, and search for matches based on several attributes.

Role: Coded the entire front-end user interaction functionality.

Tags: PHP, CMS, HTML
